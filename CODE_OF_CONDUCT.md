# Contributor Code of Conduct

## 我们的承诺

为了营造一个开放、稳定、可持续协作的社区环境，Clawdroid 项目承诺为所有贡献者提供免受骚扰的参与体验，不因年龄、体型、可见或不可见残障、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向而区别对待。

## 我们鼓励的行为

- 以尊重和事实为基础进行讨论
- 对技术方案给出清晰、可执行的反馈
- 主动补充上下文、验证步骤与边界条件
- 尊重安全问题的负责任披露流程

## 不可接受的行为

- 人身攻击、羞辱、歧视或骚扰
- 恶意挑衅、反复刷屏或破坏讨论秩序
- 在未协调的情况下公开高危利用细节或敏感数据
- 故意提交真实密钥、隐私数据或破坏性内容

## 执行责任

项目维护者有责任解释并执行本行为准则，并可对任何不当行为采取适当、公正的纠正措施。

## 适用范围

本行为准则适用于所有项目空间，包括但不限于：

- Issue
- Pull Request
- 文档讨论
- 代码审查
- 其他与项目直接相关的公开交流场景

## 报告方式

如遇到违反本行为准则的行为，请通过维护者指定的渠道私下联系。若事件同时涉及安全问题，请优先遵循 [SECURITY.md](file:///d:/temp_desktop/Proj/Clawdroid/SECURITY.md)。
